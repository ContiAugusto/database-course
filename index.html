Normalização de Tabelas

A normalização de tabelas é uma técnica utilizada no design de bancos de dados para organizar dados de forma eficiente e evitar redundâncias. Ela envolve a divisão de uma tabela grande em tabelas menores relacionadas, seguindo regras chamadas de "formas normais". Com isso, o banco de dados se torna mais fácil de gerenciar, reduzindo inconsistências e melhorando a integridade dos dados.
